### tavily_search — 互联网搜索

**功能**: 通过 Tavily API 搜索互联网，获取最新的法律新闻、判例、政策解读。

**适用场景**:
- 需要查找最新的法律政策变化或修订动态
- 本地法律数据库中没有覆盖的信息（如地方性法规、最新司法解释）
- 查找具体案例的判决结果或裁判要旨
- 了解法律实务中的最新学术观点和实务解读

**不应使用的场景**:
- 本地法律数据库已能回答的基础法律条文查询
- 不涉及时效性的通用法律概念解释

**查询优化技巧**:
- 使用中文查询以获取中文结果
- 包含时间限定词: "2025年"、"最新"
- 包含具体法律领域或关键词
- 案例搜索时包含案件类型和关键事实

**查询示例**:
- `"2025年最新个人所得税专项附加扣除政策"`
- `"最高人民法院民间借贷利率司法解释最新规定"`
- `"人工智能生成内容著作权纠纷案例"`
