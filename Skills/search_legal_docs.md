### search_legal_docs — 法律文档检索

**功能**: 搜索本地法律文档数据库，支持 BM25 + 向量多路检索 + Rerank 精排。

**适用场景**:
- 用户询问具体法律条文内容或条款解释
- 需要引用法律规定来回答问题
- 涉及法律条款的比较或冲突分析
- 案件分析中需要查找相关法条依据

**查询优化技巧**:
- 使用法律全称: "中华人民共和国刑法" 而非 "刑法"
- 包含具体条款号: "第二百六十四条"
- 包含关键法律术语: "盗窃罪"、"数额较大"、"追诉时效"
- 可组合法律名称 + 条款号 + 术语以提升检索精度

**查询示例**:
- `"中华人民共和国刑法第二百六十四条盗窃罪构成要件"`
- `"民法典合同编关于合同解除的规定"`
- `"劳动合同法经济补偿金计算标准"`
- `"公司法关于股东知情权的条款"`
